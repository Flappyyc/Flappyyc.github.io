「bzoj1047」理想的正方形

Posted on 2018-05-28

题面

Description

　　有一个ab的整数组成的矩阵，现请你从中找出一个nn的正方形区域，使得该区域所有数中的最大值和最小值的差最小。

Input

　　第一行为3个整数，分别表示a,b,n的值第二行至第a+1行每行为b个非负整数，表示矩阵中相应位置上的数。每行相邻两数之间用一空格分隔。
　 100%的数据2<=a,b<=1000,n<=a,n<=b,n<=1000

Output

　　仅一个整数，为ab矩阵中所有“nn正方形区域中的最大整数和最小整数的差值”的最小值。

Sample Input

　　5 4 2
　　1 2 5 6
　　0 17 16 0
　　16 17 2 1
　　2 10 2 1
　　1 2 2 2

Sample Output

　　1

题解

此题只要求出每个方块的大小值，然后枚举即可。每个方块的大小值，一维数组的固定窗口求最值，是单调队列的经典实例。本题就类似于二维的单调队列，对于a*b的矩阵来说，先求出每一行的固定n长度的窗口的最值。然后把求出的最值作为目标数组，再求每一列固定n长度的窗口的最值。这样就可以当作二维单调队列，题目也就获得解决。
